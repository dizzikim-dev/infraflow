## Executive Summary

경기도의회 VDI + OpenClaw + 내부망 LLM 구축을 위한 **3가지 제안 옵션**을 제시합니다.

## 옵션 A: 온프레미스 VDI

### 개요
```
┌─────────────────────────────────────────────────────────────────┐
│                        의회 내부 IDC                             │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                    DMZ 영역                              │   │
│  │   ┌─────────────┐    ┌─────────────┐                    │   │
│  │   │ VDI Gateway │    │ VDI 서버팜   │                    │   │
│  │   │ (Citrix)    │───▶│ (VM 155+)   │                    │   │
│  │   └─────────────┘    └──────┬──────┘                    │   │
│  └─────────────────────────────┼────────────────────────────┘   │
│                                │                                │
│  ═══════════════════════════════╪═══════════════════════════════│
│                                │                                │
│  ┌─────────────────────────────┼────────────────────────────┐   │
│  │                    내부망                                 │   │
│  │   ┌─────────────┐    ┌─────────────┐    ┌─────────────┐ │   │
│  │   │ LLM 서버    │◄──▶│ RAG 시스템   │◄──▶│ 문서 DB     │ │   │
│  │   │ (GPU)       │    │ (벡터DB)    │    │ (의정자료)   │ │   │
│  │   └─────────────┘    └─────────────┘    └─────────────┘ │   │
│  └──────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
```

### 구성 요소

| 구성요소 | 사양 | 수량 | 비고 |
|---------|------|------|------|
| VDI Gateway | Citrix NetScaler | 2 | HA 구성 |
| VDI 서버 | vCPU 8, 32GB RAM | 20 | 155명 수용 |
| 스토리지 | NVMe SSD 50TB | 1 | 공유 스토리지 |
| LLM 서버 | A100 80GB × 2 | 2 | 내부망 기존 활용 |
| 네트워크 | 10Gbps | - | 내부망 연결 |

### 장점
- **보안 최고 수준**: 모든 인프라가 의회 내부에 위치
- **데이터 주권 완전 확보**: 외부 전송 원천 차단
- **장기 운영 비용 절감**: 클라우드 월 비용 없음
- **네트워크 지연 최소**: 내부망 직접 연결

### 단점
- **초기 투자 비용 높음**: 하드웨어 구매 필요
- **구축 기간 길음**: 3-6개월 소요
- **확장성 제한**: 추가 장비 구매 필요
- **운영 인력 필요**: 자체 운영팀 필요

### 비용 개요 (개략)

| 항목 | 초기 비용 | 연간 운영비 |
|------|----------|------------|
| VDI 라이선스 | 3억원 | 6천만원 |
| 서버/스토리지 | 5억원 | - |
| 네트워크 | 5천만원 | 1천만원 |
| 구축/컨설팅 | 1억원 | - |
| **합계** | **9.5억원** | **7천만원** |

### 적합한 경우
- 보안이 최우선인 경우
- 장기적 관점에서 비용 최적화
- 자체 운영 역량이 있는 경우

## 옵션 B: 프라이빗 클라우드 VDI

### 개요
```
┌─────────────────────────────────────────────────────────────────┐
│                      KT 클라우드 (전용존)                         │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                    DMZ 영역                              │   │
│  │   ┌─────────────┐    ┌─────────────┐                    │   │
│  │   │ VDI Gateway │    │ VDI 서버팜   │                    │   │
│  │   │ (Cloud VDI) │───▶│ (탄력 확장)  │                    │   │
│  │   └─────────────┘    └──────┬──────┘                    │   │
│  └─────────────────────────────┼────────────────────────────┘   │
│                                │                                │
│                         전용 회선 (암호화)                        │
│                                │                                │
└────────────────────────────────┼────────────────────────────────┘
                                 │
┌────────────────────────────────┼────────────────────────────────┐
│                         의회 내부망                              │
│   ┌─────────────┐    ┌─────────────┐    ┌─────────────┐        │
│   │ LLM 서버    │◄──▶│ RAG 시스템   │◄──▶│ 문서 DB     │        │
│   └─────────────┘    └─────────────┘    └─────────────┘        │
└─────────────────────────────────────────────────────────────────┘
```

### 구성 요소

| 구성요소 | 사양 | 비고 |
|---------|------|------|
| Cloud VDI | KT Cloud DaaS | 종량제/정액제 |
| 전용 회선 | 1Gbps 암호화 | 의회-KT 직결 |
| LLM 서버 | 기존 내부망 활용 | 추가 투자 없음 |

### 장점
- **빠른 구축**: 1-2개월 내 가능
- **탄력적 확장**: 사용량에 따라 즉시 조정
- **초기 비용 낮음**: CAPEX → OPEX 전환
- **운영 부담 감소**: 클라우드 관리형

### 단점
- **월 운영비 발생**: 장기적으로 비용 증가
- **외부 의존**: 클라우드 사업자 의존
- **네트워크 지연**: 전용회선이지만 내부망보다 느림
- **보안 우려**: 일부 데이터가 외부에 위치

### 비용 개요 (개략)

| 항목 | 초기 비용 | 월 운영비 |
|------|----------|----------|
| Cloud VDI (155명) | - | 4천만원 |
| 전용 회선 | 2천만원 | 500만원 |
| 구축/컨설팅 | 5천만원 | - |
| **합계** | **7천만원** | **4.5천만원/월** |

**연간 총비용**: 7천만원 + 5.4억원 = **약 6.1억원/년**

### 적합한 경우
- 빠른 구축이 필요한 경우
- 초기 투자 여력이 제한적인 경우
- 사용량 변동이 큰 경우

## 옵션 C: 하이브리드 (PoC 권장)

### 개요
```
┌─────────────────────────────────────────────────────────────────┐
│                        의회 내부 IDC                             │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                    DMZ 영역                              │   │
│  │   ┌─────────────┐    ┌─────────────┐                    │   │
│  │   │ VDI Gateway │    │ VDI 서버     │ ← 소규모 시작     │   │
│  │   │ (경량)      │───▶│ (10-50명)   │   (확장 가능)      │   │
│  │   └─────────────┘    └──────┬──────┘                    │   │
│  └─────────────────────────────┼────────────────────────────┘   │
│                                │                                │
│  ┌─────────────────────────────┼────────────────────────────┐   │
│  │                    내부망                                 │   │
│  │   ┌─────────────┐    ┌─────────────┐                    │   │
│  │   │ LLM 서버    │◄──▶│ RAG 시스템   │ ← 기존 인프라 활용  │   │
│  │   │ (기존)      │    │ (신규)       │                    │   │
│  │   └─────────────┘    └─────────────┘                    │   │
│  └──────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
```

### 구성 요소

| 구성요소 | 사양 | 비고 |
|---------|------|------|
| VDI Gateway | 오픈소스 (Apache Guacamole) 또는 경량 상용 | 비용 최소화 |
| VDI 서버 | 기존 서버 활용 또는 최소 구매 | 10-50명 규모 |
| LLM 서버 | 기존 내부망 활용 | 추가 투자 없음 |
| OpenClaw | Docker 컨테이너 | 오픈소스 |

### 장점
- **최소 비용으로 검증**: PoC에 최적
- **빠른 시작**: 2-4주 내 가능
- **리스크 최소화**: 소규모로 시작
- **기존 인프라 활용**: 추가 투자 최소화
- **유연한 확장**: 검증 후 옵션 A/B로 전환 가능

### 단점
- **규모 제한**: 전면 도입에는 부족
- **기능 제한**: 일부 엔터프라이즈 기능 미지원
- **수동 운영**: 자동화 수준 낮음

### 비용 개요 (개략)

| 항목 | 비용 | 비고 |
|------|------|------|
| VDI 솔루션 | 1천만원 | 오픈소스 또는 경량 라이선스 |
| 서버 (필요시) | 3천만원 | 기존 활용 시 절감 |
| RAG 구축 | 2천만원 | 벡터DB + 연동 |
| 구축/컨설팅 | 2천만원 | |
| **합계** | **5천만원~8천만원** | |

### 적합한 경우
- **PoC/시범 사업** (강력 권장)
- 예산이 제한적인 경우
- 검증 후 본사업 추진 계획인 경우

## 권장 사항

### 2/13 미팅 시 제안 순서

1. **옵션 C (하이브리드)로 PoC 시작** 권장
