## 2. Phase 1: 파일럿 (10명)

### 2.1 개요

| 항목 | 내용 |
|------|------|
| **기간** | 4주 |
| **대상** | 의원 5명 + 보좌진 5명 |
| **목표** | 기능 검증, 사용자 반응 확인 |
| **예산** | 3천만원 |

### 2.2 주차별 계획

```
Week 1: 환경 구축
├── VDI 환경 설치 (경량 솔루션)
├── OpenClaw Docker 배포
├── 내부망 LLM 연동 테스트
└── 기본 RAG 구성 (회의록 100건)

Week 2: 기능 개발
├── 사용자 인터페이스 커스터마이징
├── 인증 연동 (DID/2FA)
├── 기본 프롬프트 템플릿
└── 의정자료 검색 기능

Week 3: 사용자 테스트
├── 파일럿 사용자 교육 (2시간)
├── 자유 사용 기간
├── 일일 피드백 수집
└── 버그 수정

Week 4: 평가 및 보고
├── 사용 로그 분석
├── 만족도 설문
├── 개선점 정리
└── Phase 2 계획 수립
```

### 2.3 성공 기준

| 지표 | 목표값 | 측정 방법 |
|------|-------|----------|
| 기능 동작 | 핵심 기능 100% | 기능 체크리스트 |
| 응답 시간 | < 10초 (P95) | 로그 분석 |
| 사용자 만족 | > 80% | 설문 조사 |
| 버그 | Critical 0건 | 이슈 트래커 |

### 2.4 필요 인프라

| 구성요소 | 사양 | 수량 | 비고 |
|---------|------|------|------|
| VDI 서버 | vCPU 8, 32GB | 2 | 기존 서버 활용 가능 |
| VDI Gateway | 오픈소스 | 1 | Apache Guacamole |
| LLM | 기존 내부망 | - | 추가 투자 없음 |
| 스토리지 | 1TB | 1 | 벡터DB + 문서 |

### 2.5 테스트 시나리오

```markdown
## 핵심 테스트 시나리오 (10개)

### 기본 기능
1. [ ] VDI 접속 및 인증 (DID + OTP)
2. [ ] OpenClaw UI 실행
3. [ ] 간단한 질문 응답 ("오늘 날씨")

### 의정 업무
4. [ ] 조례안 검색 ("환경 조례 현황")
5. [ ] 회의록 요약 ("지난 상임위 주요 안건")
6. [ ] 일정 조회 ("이번 주 일정")

### 보안
7. [ ] 클립보드 복사 차단 확인
8. [ ] 파일 다운로드 차단 확인
9. [ ] 세션 타임아웃 확인 (30분)

### 성능
10. [ ] 동시 10명 접속 테스트
```

## 3. Phase 2: 확장 (50명)

### 3.1 개요

| 항목 | 내용 |
|------|------|
| **기간** | 6주 |
| **대상** | 의원 25명 + 보좌진 25명 |

### 3.2 주차별 계획

```
Week 1-2: 인프라 확장
├── VDI 서버 증설
├── 스토리지 확장
├── 네트워크 최적화
└── 로드밸런서 구성

Week 3-4: 기능 고도화
├── RAG 데이터 확장 (전체 회의록)
├── 프롬프트 최적화
├── 응답 캐싱 구현
└── 사용자 피드백 반영

Week 5: 성능 테스트
├── 동시 50명 부하 테스트
├── 응답 시간 측정
├── 병목 구간 분석
└── 최적화

Week 6: 평가 및 보고
├── 성능 보고서
├── 보안 점검
├── 만족도 조사
└── Phase 3 계획
```

### 3.3 성공 기준

| 지표 | 목표값 | 측정 방법 |
|------|-------|----------|

| 응답 시간 | < 5초 (P95) | APM |
| 가용성 | > 99% | 모니터링 |
| 사용자 만족 | > 85% | 설문 조사 |
| RAG 정확도 | > 80% | 샘플 평가 |

### 3.4 추가 인프라

| 구성요소 | 사양 | 수량 | 비고 |
|---------|------|------|------|
| VDI 서버 | vCPU 8, 32GB | 5 | 신규 |
| VDI Gateway | 상용 라이선스 | 1 | 업그레이드 |
| 캐시 서버 | Redis | 1 | 응답 캐싱 |
| 모니터링 | Prometheus + Grafana | 1 | 신규 |
