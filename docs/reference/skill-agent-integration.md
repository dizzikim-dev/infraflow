## 새 기능 개발 워크플로우

1. 기능 요청 수신
2. `/brainstorm` - @pessimist + @optimist 통합 분석
3. `/plan-skill` - PR 단위 계획 수립
4. `/parallel` (해당 시) - Frontend/Backend 병렬
5. `/tdd` - RED-GREEN-REFACTOR 사이클
6. `/verify` - 테스트/빌드/디자인 검증
7. 완료 및 커밋
