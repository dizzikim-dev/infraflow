# InfraFlow 타깃 유저 분석

> 분석일: 2026-02-22
> 기반: 프로젝트 비전 + 현재 기능 수준 + 한국 시장 맥락

---

## Tier 1 — 핵심 타깃 (가장 즉각적인 가치)

### 1. 벤더 Pre-sales / 솔루션 아키텍트

| 항목 | 내용 |
|------|------|
| 누구 | Cisco, Fortinet, Palo Alto 등 벤더의 영업 기술 지원팀 |
| 현재 고통 | 고객 미팅 전 Visio/PPT로 아키텍처 제안서 그리는 데 반나절~하루 소요. 경쟁사 제품과 비교 요청 받으면 답변 어려움 |
| InfraFlow 가치 | 자연어로 즉석 다이어그램 생성 → 벤더 제품 자동 매핑 → 비용 비교까지. 고객 앞에서 실시간 데모 가능 |
| 킬러 기능 | L1(시각화) + L3(벤더 제품 추천) + 비용 비교 패널 |
| 시장 규모 | 국내 IT 벤더/리셀러 영업 인력 수천 명 |

### 2. SI(시스템 통합) 업체 아키텍트

| 항목 | 내용 |
|------|------|
| 누구 | 삼성SDS, LG CNS, SK C&C 등 SI 업체의 인프라 설계 담당 |
| 현재 고통 | 멀티벤더 환경에서 전체 아키텍처를 이해하는 사람이 없음. 시니어 엔지니어 퇴직 시 설계 지식 유실 |
| InfraFlow 가치 | 온톨로지 기반으로 "왜 이 컴포넌트가 여기 있는지" 설명. 갭 분석 + 컴플라이언스 자동 체크 |
| 킬러 기능 | L2(지식 그래프) + 갭 분석 + 컴플라이언스 리포트 |
| 시장 규모 | 국내 상위 SI 10개사, 각 수백 명의 아키텍트 |

### 3. 공공/금융 IT 담당자 (발주자)

| 항목 | 내용 |
|------|------|
| 누구 | 정부 부처, 지자체, 은행, 보험사의 IT 인프라 담당 팀장/과장 |
| 현재 고통 | 벤더가 제안하는 아키텍처가 적절한지 판단할 역량 부족. 과도한 스펙에 예산 낭비 or 부족한 설계에 보안 사고 |
| InfraFlow 가치 | "이 설계에 빠진 것은?" → 갭 분석. "PCI-DSS 준수하나?" → 컴플라이언스 체크. 벤더 독립적 검증 도구 |
| 킬러 기능 | 갭 분석 + 컴플라이언스 리포트 + 안티패턴 탐지 |
| 시장 규모 | 공공 IT 예산 연 10조+, 금융권 IT 예산 연 5조+ |

## Tier 2 — 확장 타깃 (성장 단계에서 확보)

### 4. CISO / 정보보안 책임자

| 항목 | 내용 |
|------|------|
| 누구 | 기업 정보보안 최고 책임자, 보안팀장 |
| 현재 고통 | 현재 인프라의 보안 취약점을 한눈에 파악 불가. 감사 대비 컴플라이언스 리포트 수작업 |
| InfraFlow 가치 | 아키텍처 입력 → 안티패턴 자동 탐지("방화벽 없는 DMZ") + 6개 프레임워크 컴플라이언스 리포트 자동 생성 |
| 킬러 기능 | 45개 안티패턴 탐지 + ISMS/PCI-DSS/ISO 27001 리포트 |

### 5. 클라우드 마이그레이션 컨설턴트

| 항목 | 내용 |
|------|------|
| 누구 | AWS/Azure/GCP 파트너사 컨설턴트, MSP 업체 |
| 현재 고통 | 온프레미스 → 클라우드 전환 시 어떤 서비스가 대응하는지 매핑 작업이 수작업 |
| InfraFlow 가치 | 온프레미스 아키텍처 입력 → 클라우드 서비스 자동 매핑(111개 서비스) + 비용 추정 |
| 킬러 기능 | 클라우드 추천 엔진 + 벤더↔클라우드 비교 패널 |

### 6. 인프라 교육 기관 / 대학

| 항목 | 내용 |
|------|------|
| 누구 | 네트워크/보안 교육 강사, 대학 IT학과 교수 |
| 현재 고통 | 학생들에게 아키텍처를 설명할 때 정적 PPT만 사용. 인터랙티브 실습 도구 부재 |
| InfraFlow 가치 | "3-tier 웹 아키텍처 보여줘" → 실시간 데이터 흐름 애니메이션. 패턴/안티패턴 학습 |
| 킬러 기능 | L1(애니메이션 시각화) + L2(패턴 32개 + 안티패턴 45개) |

## Tier 3 — 장기 타깃

### 7. 스타트업 CTO / 1인 DevOps

| 항목 | 내용 |
|------|------|
| 누구 | 소규모 팀에서 인프라 전체를 혼자 담당하는 엔지니어 |
| 현재 고통 | 아키텍처 설계 경험 부족. "이게 맞나?" 검증할 시니어가 없음 |
| InfraFlow 가치 | AI 시니어 아키텍트 역할 — 설계 검증, 패턴 추천, 비용 추정 |
| 필요 기능 (미구현) | 대화형 컨설팅 UI, 실시간 LLM 피드백 |

### 8. IT 미디어 / 애널리스트

| 항목 | 내용 |
|------|------|
| 누구 | Gartner, IDC 등 리서치 기관, IT 전문 미디어 |
| 현재 고통 | 벤더별 제품 비교 리포트 작성 시 데이터 수집에 수주 소요 |
| InfraFlow 가치 | 416+ 벤더 제품 + 111 클라우드 서비스 구조화된 비교 데이터 |
| 필요 기능 (미구현) | 리포트 자동 생성, 벤더 트렌드 분석 |

## 유저별 진입 시나리오

```
[가장 빠른 가치 전달]

Pre-sales 아키텍트        "다음 고객 미팅에 3-tier 웹 아키텍처 제안서 필요"
  → 프롬프트 입력 → 다이어그램 + 벤더 제품 매핑 → PDF 내보내기 (5분)

SI 설계자                "고객의 기존 인프라에서 보안 갭 찾아줘"
  → 아키텍처 입력 → 갭 분석 → 안티패턴 탐지 → 개선 제안 (10분)

공공 IT 담당자            "이 제안서가 ISMS 요건을 충족하나?"
  → 아키텍처 입력 → 컴플라이언스 리포트 자동 생성 (3분)
```

## 핵심 인사이트

| 관점 | 분석 |
|------|------|
| **가장 돈을 내줄 사람** | Pre-sales + SI 아키텍트 (제안서 생산성 = 직접적 매출 연결) |
| **가장 큰 시장** | 공공/금융 IT 발주자 (예산 규모 대, 컴플라이언스 필수) |
| **가장 빠른 도입** | 교육 기관 (무료 티어로 진입, 바이럴 효과) |
| **한국 시장 차별점** | 이중언어 + ISMS/개인정보보호법 컴플라이언스 + 국내 벤더 매핑 |
| **현재 플랫폼 준비도** | Tier 1 유저에게 80% 기능 제공 가능, Tier 3는 추가 개발 필요 |
