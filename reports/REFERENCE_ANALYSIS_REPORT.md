> 작성일: 2026-02-07

## 1. 프로젝트 의도 요약

### InfraFlow 핵심 비전
```
"프롬프트 한 줄로 인프라 아키텍처를 시각화하고,
 데이터 흐름을 애니메이션으로 보여주는 플랫폼"
```

### 핵심 기능 4가지
| 기능 | 설명 |
|------|------|
| 자연어 → 다이어그램 | 프롬프트로 복잡한 아키텍처 생성 |
| 애니메이션 데이터 흐름 | 실시간으로 데이터가 어떻게 흐르는지 시각화 |
| 보안 정책 시각화 | 방화벽, ACL, 라우팅 정책 등을 시각적으로 표현 |
| 인터랙티브 편집 | 생성된 다이어그램을 드래그앤드롭으로 수정 |

## 2. 경쟁 레퍼런스 심층 분석

### 2.1 AI 기반 다이어그램 생성 도구

#### [Eraser.io / DiagramGPT](https://www.eraser.io/diagramgpt)
**특징:**
- OpenAI 기반 자연어 → 다이어그램 변환
- 5가지 다이어그램 유형: Flow chart, ERD, Cloud Architecture, Sequence, BPMN
- Diagram-as-Code 포맷으로 렌더링 (코드/비주얼 편집 모두 지원)
- 무한 캔버스 + 실시간 협업

**가격:**
| 플랜 | 가격 | AI 크레딧/월 |
|------|------|--------------|
| Free | $0 | 5 |
| Starter | $10/user | 40 |
| Business | $25/user | 250 |

**강점:**
- 기술 팀 특화 (기술 용어 이해도 높음)
- 생각의 속도로 다이어그램 생성 가능
- SQL, Terraform 코드 붙여넣기 → 즉시 시각화

**약점:**
- Gantt 차트 등 특수 다이어그램 미지원
- GSuite/MS Office 네이티브 연동 부족
- AI 제안이 프로젝트 특성에 완벽히 맞지 않을 수 있음

---

- Draw.io 호환 (내보내기 후 편집 가능)
- 클라우드 인프라, Kubernetes, 네트워크 다이어그램 특화

**강점:**
- 러프한 아이디어/노트 → 컬러풀한 다이어그램 변환
- Draw.io 생태계 활용 가능

**특징:**
- 텍스트 프롬프트 → Flowchart, Sitemap, Mind map 등
- 다양한 다이어그램 유형 지원

---

#### [Miro AI Diagram Generator](https://miro.com/ai/diagram-ai/)
**특징:**
- 설명 → 편집 가능한 다이어그램 변환
- 화이트보드 협업 플랫폼 통합
- 초기 시각적 해석의 모호함 제거

### 2.2 Infrastructure-as-Code 시각화 도구

#### [Hava.io](https://www.hava.io/)
**특징:**
- AWS, Azure, GCP 자동 연결 → 다이어그램 자동 생성
- **버전 히스토리 & 변경 추적** (핵심 차별점)
- **보안 시각화**: Security Group, 오픈 포트, 트래픽 흐름 표시
- CI/CD 파이프라인 통합 (GitHub Actions, Terraform)

**강점:**
```
┌─────────────────────────────────────────────────────┐
│  Hava 보안 다이어그램                                │
├─────────────────────────────────────────────────────┤
│  • Security Group을 다이어그램 위에 매핑            │
│  • Ingress/Egress 포트 시각화                       │
│  • 트래픽 흐름 방향 표시                            │
│  • 취약점(잘못 구성된 접근) 즉시 식별               │
└─────────────────────────────────────────────────────┘
```

**약점:**
- Read-only 자격증명 필요 (기존 인프라만 시각화)
- 새 아키텍처 설계용으로는 부적합

---

#### [Cloudcraft](https://www.cloudcraft.co/) (Datadog 소속)
**특징:**
- AWS/Azure 라이브 환경 스캔 → 다이어그램 자동 생성
- **실시간 업데이트** (수동 재스캔 불필요)
- 2D/3D 뷰 전환
- **예산 생성기**: 아키텍처 → 자동 비용 산출

**강점:**
```
┌─────────────────────────────────────────────────────┐
│  Cloudcraft 예산 기능                                │
├─────────────────────────────────────────────────────┤
│  • 인프라 그리면서 자동으로 BOM(Bill of Materials)  │
│  • Excel 내보내기 가능                               │
│  • 멀티클라우드 비용 추정                            │
└─────────────────────────────────────────────────────┘
```

- **보안 오버레이**: 인프라 미구성 탐지 (NIST, SOC2, HIPAA 규정 기반)
- 400,000+ AWS 사용자 보유

---

**특징:**
- Terraform plan → 다이어그램 자동 생성

- **Infracost 통합**: 재정적 맥락 추가

**강점:**
- GitOps 워크플로우에 완벽 통합
- 변경 영향 즉시 시각화

---

#### [Diagrams (mingrammer)](https://github.com/mingrammer/diagrams)
**특징:**
- Python 코드로 클라우드 아키텍처 그리기

**강점:**

## 4. InfraFlow 보완점 도출

### 4.1 필수 추가 기능 (P0)

#### 1) 버전 히스토리 & 변경 추적
**레퍼런스**: Hava, Structurizr
```
현재: 버전 관리 기능 없음
권장:
├── 다이어그램 버전 자동 저장
├── 변경 사항 비교 (Diff View)
├── 특정 버전으로 롤백
└── 변경 히스토리 타임라인
```

#### 2) 실시간 협업
**레퍼런스**: Eraser, Miro, Draw.io
```
현재: 단일 사용자만 고려
권장:
├── 다중 사용자 동시 편집
├── 커서 위치 실시간 표시
├── 댓글/주석 기능
└── 공유 링크 생성
```

#### 3) 내보내기 강화
**레퍼런스**: Cloudcraft, Draw.io
```
현재: 기본 내보내기만 계획
권장:
├── PNG/SVG/PDF 고품질 내보내기
├── Terraform/CloudFormation 코드 생성
├── 프레젠테이션 모드
└── 임베드 코드 생성
```

---

### 4.2 차별화 강화 기능 (P1)

#### 4) 보안 정책 시각화 고도화
**레퍼런스**: PolicyVis, AlgoSec, Hava Security View
```
현재: 기본 정책 표시만 계획
권장 강화:
├── 정책 이상 징후 자동 탐지
├── 트래픽 흐름 시뮬레이션
├── "What-if" 분석 (정책 변경 시 영향 예측)
├── 규정 준수 체크 (ISMS, GDPR, etc.)
└── 미사용/중복 규칙 식별
```

#### 5) 애니메이션 시나리오 확장
**레퍼런스**: Structurizr Dynamic View
```
현재: 기본 데이터 흐름 애니메이션
권장 확장:
├── 장애 시나리오 시뮬레이션 (Failure Scenario)
├── 부하 분산 시각화
├── 보안 공격 경로 시뮬레이션
├── 재해 복구 시나리오
└── 성능 병목 시각화
```

#### 6) 비용 산출 기능
**레퍼런스**: Cloudcraft, Pluralith (Infracost)
```
현재: 미계획
권장 추가:
├── 아키텍처 → 예상 월간 비용 자동 산출
├── 클라우드 벤더별 비용 비교
├── 비용 최적화 제안
└── Excel/CSV 내보내기
```

### 4.3 혁신 기능 (P2)

#### 7) 라이브 인프라 연동 (선택적)
**레퍼런스**: Hava, Cloudcraft
```
권장 (장기):
├── AWS/Azure/GCP 계정 연동
├── 실시간 인프라 상태 반영
├── 설계 vs 실제 비교 (Drift Detection)
└── 모니터링 데이터 오버레이
```

#### 8) AI 대화형 수정
**레퍼런스**: Eraser AI, Miro AI
```
현재: 초기 생성만 AI
권장 확장:
├── "방화벽 추가해줘" → 기존 다이어그램에 추가
├── "DB를 이중화해줘" → 구조 변경
├── "보안 취약점 찾아줘" → 분석 후 하이라이트
└── 자연어로 지속적 수정
```

#### 9) 템플릿 마켓플레이스
**레퍼런스**: 없음 (차별화 기회)
```
권장:
├── 사전 정의 아키텍처 템플릿 (VDI, 3-Tier, etc.)
├── 커뮤니티 템플릿 공유
├── 조직 내 템플릿 라이브러리
└── 산업별/규모별 베스트 프랙티스
```
